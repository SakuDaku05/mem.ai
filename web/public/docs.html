Quick Start

Welcome to memai. This framework provides your LLM agents with semantic, causal, and procedural memory out of the box.

Installation

Install the core framework and optionally all integration dependencies using pip:

BASH
pip install memai
# Or to include all connectors:
pip install "memai[all]"
Requires Python 3.10+ and automatically sets up Kuzu (Graph) and ChromaDB (Vector) embedded engines natively.

Basic Usage (Python SDK)

You can connect to your local memory server and immediately start injecting context into your LLMs.

PYTHON
from memai.sdk import MemaiClient

# Connect to the local server
mem = MemaiClient(api_key="sk-memai-...", base_url="http://localhost:8000")

# Store memories scoped to an agent
mem.add("User prefers concise answers", agent_id="my-agent")

# Search and generate PAMI-optimized context
context = mem.inject("What does the user prefer?", agent_id="my-agent")
Docs / Getting Started / CLI Reference

CLI Reference

The memai CLI allows you to manage your memory server and databases directly from your terminal.

Commands

BASH
# Start the REST API server (auto-generates master key)
memai serve

# Add a semantic memory
memai add "Client expects weekly updates" --agent "assistant"

# Search memories (returns PAMI context)
memai search "updates" --agent "assistant"

# Run a staleness pruning sweep
memai sweep
Docs / Core Concepts / PAMI Injection

Position-Aware Memory Injection

LLMs suffer from the "lost in the middle" phenomenon—they pay attention to the beginning and end of their prompt, but ignore the middle.

PAMI solves this by dynamically partitioning high-utility memories. When you call mem.inject(), the framework calculates a utility score (Q-score) for every relevant fact, and places the absolute highest-scoring facts at the very top and very bottom of the injected text block.

The mem.inject() method automatically applies PAMI based on the configured context_budget. No manual array sorting required.
Docs / Core Concepts / Staleness Rules

Staleness Rules (R1-R4)

Unlike standard vector databases that blindly accumulate data until they hallucinate, `memai` actively prunes itself.

  • R1 (Time Decay): Memories lose utility exponentially over time unless frequently accessed.
  • R2 (Superseded Facts): When a direct update occurs ("I moved to LA"), the previous contradictory fact ("I live in NYC") is flagged as stale.
  • R3/R4 (Graph Culling): Orphaned events in the Kuzu graph are vacuumed to save memory.
Docs / Integrations / LangChain

LangChain Connector

Use MemaiMemory as a drop-in replacement for standard ConversationBufferMemory.

PYTHON
from memai.connectors.langchain import MemaiMemory
from langchain.chains import ConversationChain

memory = MemaiMemory(api_key="sk-memai-...", agent_id="lc-agent")
chain = ConversationChain(llm=llm, memory=memory)
Docs / Integrations / Claude Code

Claude Code (MCP Server)

memai exposes a Model Context Protocol (MCP) stdio server for native integration with Anthropic's Claude Code.

Add the following to ~/.config/claude/mcp_servers.json:

JSON
{
  "memai": {
    "command": "python",
    "args": ["-m", "memai.connectors.mcp"],
    "env": {
      "MEMAI_API_KEY": "sk-memai-...",
      "MEMAI_AGENT_ID": "claude-code"
    }
  }
}

OpenAI Auto-Wrapper

Intercepts the standard OpenAI SDK to inject memory and save outputs completely invisibly.

PYTHON
from openai import OpenAI
from memai.connectors.openai import wrap_openai

client = wrap_openai(OpenAI(), api_key="sk-...")

# Automatically retrieves memory for 'user-1' and saves the response
res = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "What's my favorite food?"}],
    user="user-1" 
)
