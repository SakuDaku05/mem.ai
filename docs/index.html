Proven Academic Performance

Engineered against the strict standards of the BEAM (ICLR 2026) benchmark. By solving the "lost-in-the-middle" phenomenon using Position-Aware Memory Injection (PAMI), `memai` significantly outperforms standard RAG baselines in temporal reasoning, information extraction, and conflict resolution.

Our auto-pruning (R1-R4) algorithms also ensure that context windows remain clean, preventing the LLM from hallucinating on outdated, superseded facts.

Everything you need to give your AI assistants long-term, human-like recall.

Semantic Retrieval

Ultra-fast dense vector retrieval powered by an embedded ChromaDB engine. Store millions of facts, user preferences, and snippets with absolute multi-tenant agent isolation.

🕸️

Causal Event Graph

True temporal reasoning. Our native Kuzu-powered graph database tracks not just what happened, but how events cause and precede one another in time.

🎯

PAMI Injection

Position-Aware Memory Injection calculates the utility (Q-score) of every fact and structurally forces critical context to the boundaries of the LLM window to guarantee high recall.

🧹

Auto-Pruning Dynamics

Unlike standard RAG that blindly accumulates data, `memai` actively prunes itself. Built-in algorithms detect staleness, decay older memories, and silently cull direct contradictions.
