Architecture

Six subsystems, one interface

Every layer of memory an agent needs — unified behind a single Memory orchestrator.

🕸️

EventMemory

Kuzu-backed causal graph. Stores events as nodes with typed edges (CAUSES, PRECEDES, CONTRADICTS). Enables temporal reasoning and causal chain traversal.

Kuzu embedded graph
📚

SemanticMemory

ChromaDB vector store for facts, preferences, and summaries. Dense retrieval with cosine similarity and agent-scoped isolation.

ChromaDB · all-MiniLM-L6-v2
🔁

ProceduralMemory

SQLite-backed workflow store. Named procedures with trigger patterns and ordered steps. Regex + keyword matching for automatic replay.

SQLite · regex matching
🧹

StalenessDetector

Four rule-based pruning rules: time decay (R1), superseded facts (R2), contradiction detection (R3), dormant memories (R4).

R1–R4 rules
⚡

UtilityScorer

Q-value-inspired composite scoring: recency, access frequency, cosine similarity, causal distance. Re-ranks retrieved memories before injection.

Composite Q-scoring
🎯

PAMI

Position-Aware Memory Injection. High-utility memories go at the start and end of context — solving the lost-in-the-middle problem with configurable token budgets.

Boundary injection · token budget
Quick Start

Up and running in 60 seconds

Install, serve, and start storing agent memories with a three-line SDK call.

pip install memai
Python SDK
CLI
REST API
from memai.sdk import MemaiClient

# Connect to your memai server
mem = MemaiClient(api_key="sk-memai-...", base_url="http://localhost:8000")

# Store memories
mem.add("User prefers concise answers", agent_id="alice")
mem.add("User's favourite language is Python", agent_id="alice")

# Search + inject into LLM (PAMI-optimised)
context = mem.inject("What does Alice prefer?", agent_id="alice")
messages = [
    {"role": "system", "content": f"You are helpful.\n\n{context}"},
    {"role": "user", "content": user_input},
]

# Session-scoped causal event tracking
with mem.session("alice") as s:
    e1 = s.add_event("User submitted form")
    e2 = s.add_event("Validation failed", caused_by=e1.id)
    chain = s.causal_chain(e1.id)
